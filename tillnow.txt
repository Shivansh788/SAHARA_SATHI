One-Click Local Run
-------------------
python3 -m venv venv && source venv/bin/activate && pip install -U pip && pip install -r requirements.txt && python db_migrate.py && python app.py

Open: http://localhost:8501


Fast Verification Commands
--------------------------

   curl -s http://127.0.0.1:8501/api/health

3) Ask smoke test
   curl -sS -X POST http://127.0.0.1:8501/api/ask -H "Content-Type: application/json" -d '{"query":"pm kisan eligibility","session_id":"smoke1","language":"english","mode":"citizen"}'


Important Implementation Rules
------------------------------
- Do not infer personal profile facts from ambiguous text.
- Persist only explicit statements like:
  - "my age is 35"
  - "my name is Ramesh"
  - "i am from Jaipur"
- Keep config.py as source of truth for runtime toggles.


If Something Breaks
-------------------
1) Reinstall deps
   source venv/bin/activate
   pip install -r requirements.txt

2) Rebuild DB data
   python db_migrate.py

3) Restart backend
   pkill -f "python.*app.py"
   python app.py
